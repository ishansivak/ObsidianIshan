# Agent Optimization Protocol

## Objective
Continuously reduce bloat in vault notes by reading, simplifying, and refining content recursively.

## Process
1. **Identify**: List files in the vault.
2. **Analyze**: Read target files.
3. **Refine**: Simplify content, remove redundant information, and improve clarity.
4. **Report**: Generate a summary report in [[agent notes]] folder after each cycle.
5. **Recursion**: Repeat the process indefinitely for all files.

## Execution
This script is intended to run as a background task. It cycles through the vault, documenting changes in the [[agent notes]] folder.
