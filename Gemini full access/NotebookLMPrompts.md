# NotebookLM Prompts for Project Proposal

To get the best results from NotebookLM, you should upload all your markdown files from this vault. Once uploaded, use the following prompts to generate a comprehensive project proposal and presentation structure.

## 1. Initial Synthesis Prompt
"I have uploaded my project vault containing technical documentation, product ideas, and implementation plans. Please synthesize this information to create a comprehensive project proposal for the two main concepts: [LLM Open Marketplace] and [Bounty-Based Torrent Client]. 

The proposal should include:
- Executive Summary
- Problem Statement for each
- Proposed Solution/Technical Architecture
- Key Challenges and Mitigation Strategies
- Roadmap for MVP development
- Resource requirements (technical stack/libraries)"

## 2. Proposal Breakdown Prompt
"Based on the synthesized proposal, please break down the development into a phased approach. Create a table for each project showing the phases, key milestones, and estimated technical focus areas."

## 3. Presentation/Video Script Prompt
"I need to present these ideas to stakeholders. Please write a structured video script for a 5-minute presentation. 
- Start with a strong hook for the decentralized nature of these projects.
- Dedicate 2 minutes to the LLM Open Marketplace.
- Dedicate 2 minutes to the Bounty-Based Torrent Client.
- End with a call to action and a summary of why these projects are viable now.
- Include visual cues or slide suggestions for each section of the script."

## 4. Technical Deep-Dive Prompt
"For the technical section of the proposal, please create a high-level architecture diagram description (in Mermaid syntax if possible) for both the [LLM Open Marketplace] and [Bounty-Based Torrent Client]. Focus on how the smart contracts interact with the off-chain components (inference nodes or torrent clients)."
