# Milk-V Duo 256M

The Milk-V Duo 256M is a compact, cost-effective RISC-V SoC board capable of running embedded Linux. It is well-suited for edge computing tasks, including audio processing and sensor data aggregation due to its low power consumption and versatile I/O.

## Applications
- [[AI Voice Recorder Node]]
- Sensor Data Collection
- Edge AI Processing

## Key Features
- RISC-V architecture
- Embedded Linux support
- Compact form factor
- GPIO and peripheral support for sensors and audio modules
