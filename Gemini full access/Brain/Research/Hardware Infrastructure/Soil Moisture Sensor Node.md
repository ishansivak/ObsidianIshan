# Soil Moisture Sensor Node

## Overview
Sensor nodes monitoring environmental parameters in agricultural settings, acting as the physical interface between the protocol layer and the biological substrate.

## Components
- **Mechanism**: Capacitive soil moisture sensors (prevents corrosion).
- **Controller**: ESP32 (deep-sleep, Wi-Fi/Bluetooth edge computing).
- **Power**: Solar-harvested LiPo battery system for autonomous operation.

## Data Integration
Captured data is ingested by [[Saito Consensus and Operations]]. Mapping moisture data against [[Fungi and Mycelial Network Systems]] activity models the correlation between environmental moisture and mycelial growth, creating a feedback loop between physical, biological, and protocol layers.
