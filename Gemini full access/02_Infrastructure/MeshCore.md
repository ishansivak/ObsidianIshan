# MeshCore

MeshCore is designed for structured, high-capacity, permanent mesh infrastructure.

## Core Architecture
- **Routing**: Role-based relay routing.
- **Topology**: Planned infrastructure using fixed, high-elevation "Repeaters."
- **Constraints**: Supports up to 64 hops due to efficient traffic management.
- **Use Cases**: Municipal-scale backbones, emergency services, agricultural monitoring.

## Pros & Cons
- **Pros**: Lower latency, longer battery life for end devices, manages larger, more stable networks.
- **Cons**: Requires pre-planned infrastructure deployment.
