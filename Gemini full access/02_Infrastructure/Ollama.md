# Ollama

Ollama is a tool for running large language models (LLMs) locally on your own hardware. 

## Key Features
- Supports popular models like Llama 3, Mistral, and others.
- Provides a simple API for local inference.
- Highly relevant for decentralized or privacy-focused AI applications.

## Integration
- This is a core component for the [[LLM Open Marketplace]] concept.
- Can be used in conjunction with [[Mesh Networking]] nodes for local, edge-based AI processing.
