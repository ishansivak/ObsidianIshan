# Athena: Glassmorphic Theme Variables

## Core Variables
- `bg-glass`: `bg-white/10` (backdrop-blur-md)
- `border-glass`: `border-white/20`
- `shadow-glass`: `shadow-lg`

## Implementation
- Apply `bg-glass` to all primary containers.
- Apply `border-glass` to all container edges.
- Apply `shadow-glass` to all floating elements.
