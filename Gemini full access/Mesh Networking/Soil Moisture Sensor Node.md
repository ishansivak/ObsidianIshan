# Soil Moisture Sensor Node

This device monitors soil volumetric water content, allowing for precision irrigation.

## Connectivity
- **Radio**: [[LoRa]] (integrated with [[Meshtastic]] or [[MeshCore]])
- **Communication**: Sends periodic telemetry data to the central hub via the [[Mesh Networking]] infrastructure.

## Technical Specifications
- **Sensor Type**: Capacitive soil moisture sensor
- **Power**: Solar-powered LiPo battery
- **Data Interval**: Configurable (default: 15 minutes)

## Applications
- Automated irrigation scheduling
- Drought stress monitoring
- Crop water-use efficiency analysis
